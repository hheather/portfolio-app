Meta Memory

Meta Memory is a speculative VR installation exploring what digital spaces could look like if we as a society continue to hoard digital waste.

By drawing comparions between computer and human memory, the piece integrates publicly-sourced 3D scans as real-world examples of digital waste. On interaction with each object, they attempt to reason with the user for their continued existence, in the form of a generated poem via the Deepseek API. The more the user interacts, the more the world degrades, making visible the hidden costs of energy consumption in real-life digital platforms.

Exhibited at Peckham Digital, 2025.

Tools & Technologies

  • Unreal Engine 5
  • Virtual Reality
  • Deepseek API
  • Meta Quest 3

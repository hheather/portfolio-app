Meta Memory

Meta Memory is a speculative VR installation exploring what digital spaces could look like if we as a society continue to hoard digital waste.

By drawing comparions between computer and human memory, the piece integrates publicly-sourced 3D scans as real-world examples of digital waste. On interaction with each object, they attempt to reason with the user for their continued existence, in the form of a generated poem via the Deepseek API. The more the user interacts, the more the world degrades, making visible the hidden costs of energy consumption in real-life digital platforms.

Exhibited at Peckham Digital, 2025.

Tools & Technologies

  • Unreal Engine 5
  • Virtual Reality
  • Deepseek API
  • Meta Quest 3
  • Worldbuilding
  • Polycam
  • Every part of the piece was meticulously designed with ease of interaction in mind, due to the short-form installation experience we wanted to provide for an exhibition context.
  • From the user's entry into the world, they are greeted with a simple instruction on how to navigate, with a hint about how to interact with the objects.
  • The user navigates simply with the toggle switch and a simple button press for object-interaction.
  • On each interaction, the position of the sun in the sky is altered, utilising Unreal Engine 5's built-in lighting capabilities to create a stunning sunset, symbolising the deteriation of the world.
  • On the final interaction, the sun fully sets, and an info screen appears to signal the installation's end.

Learnings

I'd never built anything before with Unreal Engine or Virtual Reality, so there were definitely some steep learning curves. I found Blueprints/visual coding fairly easy to get my head around due to my background in software development, but converting the installation from PC into a VR-playable experience was the tricky part due to the sheer size of the installation. To combat this, various optimisations needed to be made to reduce its size. For the exhibition, we ran it from a gaming PC which gave us enough compute for a seamless VR experience.
